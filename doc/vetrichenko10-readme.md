## Мета
Розробка параметризованих методів (Generic Methods) та реалізація автоматичного режиму роботи програми.

## 1 ВИМОГИ
1. Реалізувати параметризований метод сортування, що приймає `Comparator`.
2. Реалізувати сортування Адресної книги за:
    - Прізвищем;
    - Ім'ям;
    - Датою народження;
    - Датою редагування.
3. Реалізувати два режими роботи:
    - Діалоговий (Меню);
    - Автоматичний (ключ `-auto`).

## 2 ОПИС ПРОГРАМИ

### 2.1 Generic Method (ContainerTools)
Створено клас `ContainerTools`, який містить статичний параметризований метод:
```java
public static <T> void sort(LinkedContainer<T> container, Comparator<T> comparator)
```
