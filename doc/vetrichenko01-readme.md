### 1.3 Задача (Варіант №3)
1. Ініціалізувати змінні, використовуючи літерали:
    - Номер заліковки (рандом `54321`) -> **HEX** (`0xD431`).
    - Телефон (`380962849509`) -> **DECIMAL** (`...L`).
    - Останні 2 ненульові цифри телефону (`59`) -> **BINARY** (`0b111011`).
    - Останні 4 ненульові цифри телефону (`8459`) -> **OCTAL** (`020413`).
2. Обчислити символ: номер 3 -> Літера **'C'**.
3. Підрахувати кількість парних/непарних цифр та одиниць у бітовому представленні для кожного числа.

### 2.2 Літерали
Використано 4 типи літералів для запису цілих чисел:
```java
int studentId = 0xD431;      // Шістнадцятковий (Hex)
long phoneNumber = ...L;     // Десятковий Long
int lastTwoDigits = 0b111011;// Двійковий (Binary)
```
