## Мета
Використання стандартних колекцій Java для зберігання та обробки об'єктів.

## 1 ВИМОГИ
1. Замінити власні контейнери на стандартні класи з пакету `java.util`.
2. Реалізувати сортування за допомогою компараторів (з Лаб 10).
3. Забезпечити збереження колекції у файл (XML).
4. Реалізувати пошук, додавання та видалення елементів.

---

## 2 ОПИС ПРОГРАМИ

### 2.1 Використання JCF
Замість класу `LinkedContainer` використано інтерфейс `java.util.List` та його реалізацію `java.util.ArrayList`.
Це надає готові методи:
- `add(element)` — додавання.
- `remove(index)` — видалення.
- `get(index)` — доступ.
- `size()` — кількість.
- `forEach(...)` — ітерація.

### 2.2 Сортування
Для сортування використано стандартний метод інтерфейсу List:
```java
list.sort(BY_SURNAME);
```
