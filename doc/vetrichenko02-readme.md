### 1.2 Загальне завдання
Розробити програму, що обробляє 10 випадкових чисел. Використовувати клас `Random`. Заборонено використовувати `String` для обробки даних та масиви. Результат вивести у вигляді таблиці.

### 1.3 Задача (Варіант №3)
Знайти найбільшу цифру в десятковому запису цілочисельного значення.



## 2 ОПИС ПРОГРАМИ

### 2.1 Алгоритм (Декомпозиція)
Задача розбита на два методи:
1.  `main()`: Відповідає за генерацію чисел, організацію циклу та виведення таблиці.
2.  `findMaxDigit(int number)`: Реалізує логіку пошуку.
    -   Береться модуль числа (`Math.abs`).
    -   У циклі `while (number > 0)` отримується остання цифра операцією `% 10`.
    -   Цифра порівнюється зі змінною `max`.
    -   Число зменшується у 10 разів (`/ 10`).

### 2.2 Структура
Пакет: `ua.khpi.oop.vetrichenko02`
Клас: `Main`

### 2.3 Важливі фрагменти коду
```java
while (temp > 0) {
    int digit = temp % 10; // Беремо останню цифру
    if (digit > max) {
        max = digit;       // Оновлюємо максимум
    }
    temp /= 10;            // Відкидаємо останню цифру
}
```
