<p align="center"><b>Тема: Основи введення/виведення Java SE (Long Term Persistence)</b></p>

## Мета
Оволодіння навичками управління введенням/виведенням даних, використання `XMLEncoder` та навігація по файловій системі.

## 1 ВИМОГИ
1. Реалізувати збереження масиву об'єктів (Адресна книга) у форматі XML.
2. Використати механізм **JavaBeans Long Term Persistence** (`XMLEncoder`/`XMLDecoder`).
3. Заборонено використовувати стандартну серіалізацію (`ObjectOutputStream`).
4. Реалізувати інтерактивний вибір директорії для збереження файлу (навігація по папках).

---

## 2 ОПИС ПРОГРАМИ

### 2.1 Long Term Persistence
На відміну від бінарної серіалізації, використано класи `java.beans.XMLEncoder` та `XMLDecoder`.
Це дозволяє зберігати стан об'єкта у текстовому форматі XML, який є зрозумілим для людини та може бути відредагований вручну.

Приклад збереження:
```java
XMLEncoder encoder = new XMLEncoder(new BufferedOutputStream(new FileOutputStream(file)));
encoder.writeObject(book); // Запис об'єкта
encoder.close();
```
