## Мета
Розробка власних утилітарних класів. Набуття навичок розв'язання прикладних задач з використанням масивів і рядків.

### 1.2 Загальне завдання
Розробити консольну програму, використовуючи власний утилітарний клас зі статичними методами. Продемонструвати роботу з `StringBuilder`. Заборонено використовувати регулярні вирази.

### 1.3 Задача (Варіант №3)
Ввести декілька рядків. Розбити на дві групи:
1. Рядки, довжина яких менша за середню.
2. Рядки, довжина яких не менше середньої.
   Вивести рядки та їх довжину по групах.

## 2 ОПИС ПРОГРАМИ

### 2.1 Структура
Програма складається з двох класів:
1.  **Main:** Клас-драйвер, який ініціалізує масив рядків і викликає методи обробки.
2.  **StringHelper:** Власний утилітарний клас.
    -   Конструктор `private` (заборона створення екземпляра).
    -   Методи `static` (можна викликати без створення об'єкта).

### 2.2 Використання StringBuilder
Згідно з вимогами, для формування текстового звіту використано клас `StringBuilder`, що дозволяє ефективно конкатенувати рядки без створення зайвих об'єктів у пам'яті.

### 2.3 Важливі фрагменти коду (StringHelper)
```java
// Розрахунок середнього арифметичного
public static double calculateAverageLength(String[] strings) {
    int totalLength = 0;
    for (String s : strings) {
        totalLength += s.length();
    }
    return (double) totalLength / strings.length;
}

// Фільтрація та використання StringBuilder
public static String getGroupReport(...) {
    StringBuilder sb = new StringBuilder();
    // ... логіка перевірки ...
    sb.append("Result string...");
    return sb.toString();
} 
```
