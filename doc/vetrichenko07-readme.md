## Мета
Використання об'єктно-орієнтованого підходу для розробки об'єкта предметної галузі.

## 1 ВИМОГИ
### 1.1 Розробник
- **Автор:** Ветріченко Дарина
- **Група:** КН-924з
- **Варіант:** 3 (Адресна книга)

### 1.2 Завдання
Спроектувати клас `AddressEntry` (Запис адресної книги) з полями:
- П.І.Б.
- Дата народження
- Телефони (список)
- Адреса
- Дата і час редагування (автоматично).
  Забезпечити підтримку кирилиці та управління масивом об'єктів.

---

## 2 ОПИС ПРОГРАМИ

### 2.1 Domain-об'єкт (AddressEntry)
Клас спроектовано за принципами **JavaBeans**:
1.  Усі поля мають модифікатор доступу `private` (Інкапсуляція).
2.  Доступ до полів здійснюється через публічні методи `get` та `set`.
3.  Поле `phones` реалізовано як `ArrayList<String>`, що дозволяє зберігати необмежену кількість номерів.
4.  Поле `lastEditTime` типу `LocalDateTime` оновлюється автоматично в кожному сеттері (`updateTimestamp()`), що фіксує реальний час зміни даних.

### 2.2 Підтримка кирилиці
Для коректного відображення українських літер у консолі використано налаштування потоку виведення:
```java
System.setOut(new PrintStream(System.out, true, StandardCharsets.UTF_8));
```
