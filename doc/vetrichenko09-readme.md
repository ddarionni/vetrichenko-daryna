## Мета
Розробка параметризованих класів та методів. Створення власного контейнера на основі зв'язного списку.

## 1 ВИМОГИ
1. Створити клас-контейнер `LinkedContainer<T>`, використовуючи Generics.
2. Реалізувати структуру даних **Зв'язний список** (Linked List).
3. Забезпечити підтримку циклу `for-each` (інтерфейс `Iterable`).
4. Реалізувати збереження даних через серіалізацію та XML.
5. Заборонено використовувати стандартні `java.util.LinkedList`.

---

## 2 ОПИС ПРОГРАМИ

### 2.1 Параметризований клас LinkedContainer<T>
Замість масиву, контейнер використовує динамічну структуру вузлів:
```java
private static class Node<E> {
    E element;
    Node<E> next;
}
```
## ВИСНОВКИ
У ході роботи було розроблено власний параметризований контейнер на базі однозв'язного списку.

Опановано використання Generics (<T>) для створення універсальних класів.

Реалізовано алгоритми додавання та видалення елементів у зв'язному списку (робота з посиланнями next).

Забезпечено повну сумісність з механізмами серіалізації Java.
