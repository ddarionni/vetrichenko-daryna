<p align="center"><b>Лабораторна робота №4</b></p>
<p align="center"><b>Тема: Інтерактивні консольні програми для платформи Java SE</b></p>

## Мета
Реалізація діалогового режиму роботи з користувачем та обробка параметрів командного рядка.

## 1 ВИМОГИ

### 1.1 Розробник
- **Автор:** Ветріченко Дарина
- **Група:** КН-924з
- **Варіант:** 3

### 1.2 Завдання
Модифікувати програму Лабораторної роботи №3. Додати:
1.  Текстове меню (введення, перегляд, розрахунок, вихід).
2.  Обробку параметрів запуску:
    - `-h`: вивід довідки.
    - `-d`: режим Debug (додаткові повідомлення в консолі).

---

## 2 ОПИС ПРОГРАМИ

### 2.1 Структура
- `Main`: відповідає за взаємодію з користувачем. Містить цикл `while` для меню та `switch` для вибору команд.
- `StringHelper`: утилітарний клас (з попередньої роботи) для логіки розрахунків.

### 2.2 Обробка параметрів
При старті `main(String[] args)` перевіряється масив `args`.
```java
if (arg.equals("-d")) {
    debugMode = true; // Вмикає вивід діагностичних повідомлень
} 

```
## ВИСНОВКИ
У ході роботи було додано інтерактивність до консольної програми.

Реалізовано обробку параметрів командного рядка для керування режимами роботи.

Створено зручне текстове меню для багаторазового введення та обробки даних без перезапуску програми.

Застосовано клас Scanner для введення даних з клавіатури.
